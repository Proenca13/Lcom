int check_can_flag ( )

Checks if more flags can be placed.

This function counts the current number of flagged cells and checks if it is less than the allowed maximum (10 flags). If the number of flags is less than 10, more flags can be placed.

Returns
int 1 if more flags can be placed, 0 otherwise.

int check_first_touch ( )

Checks if the first cell has been touched.

This function iterates through the grid to check if any cell has been revealed. If no cell is revealed, it indicates the first touch has not been made.

Returns
int 1 if no cell has been revealed, 0 otherwise.
